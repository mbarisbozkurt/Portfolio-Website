Hand Gesture-Controlled Music Player

    Made with:

A music player application that integrates with Spotify and can be controlled through hand gestures. The computer camera detects user's hand movements in real-time and converts these movements into music controls. Users can play and pause songs and scroll through their playlist using hand gestures. Music controls can be performed both through the Spotify application and hand gestures, with both applications working in synchronization. It is an application that offers a contactless and innovative user experience by replacing traditional music player controls with hand gestures.

Multi-Purpose Raman Spectrometer

    Made with:

This TÜBİTAK-supported project focuses on the development of both hardware and software for a Raman spectrometer, an analytical instrument that uses laser light to analyze the molecular composition of materials. As a 5-member student team, we contributed to the software development of this project over a 3-month period as part of our graduation project. The software includes Python-based data processing tools, such as smoothing, baseline correction, and peak detection. Built with React.js and Electron, it features an intuitive UI design, authentication, and ongoing implementation of real-time data processing. While our contribution focused on the software aspect, the project as a whole is still under active development, and the GitHub code remains private.
